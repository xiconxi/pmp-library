Showcase: Interactive HTML Slides

02 Jun 2019

One of the nifty features of the pmp-library is cross-compilation to JavaScript, thereby allowing to run mesh processing applications directly in a WebGL2-compatible browser. This feature also allows to seamlessly include demo applications into HTML lecture slides, allowing students to interactively explore geometry processing algorithms. We just published a set of example slides demonstrating this application case:
